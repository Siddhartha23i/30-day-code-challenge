# 30-Day Code Every Day Challenge

I'm Siddharth — a final-year CSE (AI & ML) student.
This repo is my proof of work.

---

## Why I'm doing this

I got tired of planning and not doing.  
So starting today, for 30 days straight — I open a problem, I solve it, I push it. No excuses.

The goal isn't to become a genius in 30 days.  
The goal is to build the habit of showing up — every single day.

---

## What this challenge includes

- **1 LeetCode problem daily** — easy to medium, focused on understanding patterns not just solving
- **1 concept daily** — rotating between SQL, Backend (FastAPI / Node.js), or Python fundamentals
- All solutions and notes pushed to this repo before midnight
